New in v0.2.0

Meta-RAG: Context-Aware Intelligence

Stop flooding your context window with raw code. CodeFlow's Meta-RAG uses hierarchical summarization to retrieve exactly what's needed, reducing token usage by 80% while improving accuracy.

Hierarchical Summaries

Maintains concise summaries at file, class, and function levels to enable drill-down retrieval.

State-of-the-Art Accuracy

Achieves 84.67% file-level bug localization accuracy on SWE-bench Lite, outperforming traditional RAG.

Cost Efficient

Drastically reduces API costs by sending compact summaries instead of full source files.

Meta-RAG Process
1
Query
"Fix the auth validation bug"
2
Retrieve Summaries
File: auth.py (Summary: Handles user auth...)
  Class: Auth (Summary: Main auth logic...)
    Func: validate (Summary: Validates tokens...)
3
Targeted Retrieval
Agent selects auth.py:validate for full context.
